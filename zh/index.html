ComfyUI-TD 通过更稳定的 WebSocket 工作流，把 图像、视频、音频、 点云 和 Gaussian Splatting PLY 从 ComfyUI 发送到 TouchDesigner。

核心功能

ComfyUI-TD 的重点是把 ComfyUI 生成数据稳定地实时传输到 TouchDesigner。

图像工作流

将 ComfyUI 生成图像传给 TouchDesigner，并解析为 TOP 流程。

视频与音频

支持视频和音频输出，适合实时视觉、播放和交互式系统。

3D 与 Gaussian Splatting

支持传统点云和 Gaussian Splatting PLY 在 TouchDesigner 中使用。

ComfyUI-TD 需要与 ComfyUI2TD.tox 配合使用，建议同步更新到较新的 tox 版本。

支持的数据类型

  • ComfyUI 生成图像
  • 用于 TouchDesigner 实时播放的视频数据
  • 适用于 CHOP 工作流的音频输出
  • 传统 PLY 点云数据
  • Gaussian Splatting PLY 数据

典型使用场景

  • 基于 ComfyUI 与 TouchDesigner 的互动装置
  • 云端 ComfyUI 输出回传到本地 TD
  • 结合 AI 生成和实时视觉的技术美术流程
  • 点云、3D 与 Gaussian Splatting 的可视化系统

安装方式

方式 1：ComfyUI-Manager

在 ComfyUI-Manager 中搜索 ComfyUI-TD 并直接安装。

方式 2：手动安装

将项目复制到 ComfyUI/custom_nodes 目录。

方式 3：Git Clone

cd custom_nodes
git clone https://github.com/JiSenHua/ComfyUI-TD.git

使用说明

  • 部分节点基于 ComfyUI-Tooling-Nodes 进行了移植和优化。
  • 重写后的 WebSocket 接口更适合弱网络环境下的数据返回。
  • 较新版本的预设工作流已经不再依赖 ComfyUI-Tooling-Nodes。
  • 如果你使用远程服务器运行 ComfyUI，也可以走云端传输方案。

常见问题

ComfyUI-TD 是什么？

它是一个将 ComfyUI 生成媒体和 3D 数据实时传输到 TouchDesigner 的自定义节点。

是否必须配合 ComfyUI2TD.tox？

是的。ComfyUI-TD 需要和 ComfyUI2TD.tox 一起使用。

是否支持视频、音频和点云？

支持。当前支持图像、视频、音频、传统点云 PLY 和 Gaussian Splatting PLY。

有没有英文页面给 Google/Bing 使用？
