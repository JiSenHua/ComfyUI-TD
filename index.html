ComfyUI-TD connects ComfyUI and TouchDesigner through a more stable WebSocket workflow for images, video, audio, point clouds, and Gaussian Splatting PLY.

The project is built for creators, technical artists, and developers who want a practical bridge between AI generation pipelines and real-time media systems.

Why people search for ComfyUI-TD

ComfyUI-TD is used for TouchDesigner integration, cloud ComfyUI workflows, WebSocket media streaming, and 3D or Gaussian Splatting previews inside TD.

Core features

ComfyUI-TD focuses on reliable real-time data transmission from ComfyUI into TouchDesigner.

Image workflows

Send generated images to TouchDesigner and parse them into TOP workflows.

Video and audio

Transfer video and audio output for live visuals, playback, and generative systems.

3D and Gaussian Splatting

Support traditional point clouds and Gaussian Splatting PLY workflows in TouchDesigner.

ComfyUI-TD must be used together with ComfyUI2TD.tox. Keeping the tox component updated is recommended.

Supported data types

  • Images from ComfyUI generation pipelines
  • Video data for live playback inside TouchDesigner
  • Audio output for CHOP-based workflows
  • Traditional PLY point cloud data
  • Gaussian Splatting PLY data

Typical use cases

  • Interactive installations powered by ComfyUI and TouchDesigner
  • Cloud-based ComfyUI output streamed back to local TD systems
  • Technical art pipelines that mix AI generation with real-time visuals
  • 3D preview and point-cloud-based media systems

Installation options

Method 1: ComfyUI-Manager

Search for ComfyUI-TD in ComfyUI-Manager and install it directly.

Method 2: Manual install

Copy the project into ComfyUI/custom_nodes.

Method 3: Git clone

cd custom_nodes
git clone https://github.com/JiSenHua/ComfyUI-TD.git

Workflow notes

  • Some nodes are ported and optimized from ComfyUI-Tooling-Nodes.
  • The rewritten WebSocket interface improves return stability in weak network conditions.
  • Preset workflows from newer versions no longer depend on ComfyUI-Tooling-Nodes.
  • Cloud deployment is supported for users who run ComfyUI remotely.

FAQ

What is ComfyUI-TD?

It is a ComfyUI custom node that sends generated media and 3D data into TouchDesigner in real time.

Do I need ComfyUI2TD.tox?

Yes. ComfyUI-TD is designed to work together with the ComfyUI2TD.tox component.

Does the project support video, audio, and point clouds?

Yes. The project supports image, video, audio, point cloud PLY, and Gaussian Splatting PLY workflows.
